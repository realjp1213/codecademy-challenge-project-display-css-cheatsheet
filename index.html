CSS Box Model Display & Positioning Table

display Property

This CSS property allows you to control how an element flows vertically and horizontally in a document.

Next, we'll explore the values that can be used with display and how the declaration will impact how elements are positioned on the browser window.

display property values

Value Description
position Defines how elements are placed within the document.
relative Positions an element relative to its normal document flow placement, allowing for improved styling control.
absolute Places an element relative to its nearest positioned ancestor or the browser window.
fixed Positions an element relative to the viewport, removing it from the document flow and keeping it fixed during page scrolling.
sticky Element sticks to a defined offset position when the user scrolls its parent container.
z-index Specifies how far back or how far forward an element appears on the page when it overlaps other elements.
inline Elements take up as little space as possible, and they cannot have manually adjusted width or height
block Elements take up the width of their container and can have manually adjusted heights
inline-block Elements have set width and height, but they can also appear next to each other and do not take up their entire container width.

Other Positioning Properties

float & clear

Value Description
float Positions an element to the left or right of its container, allowing other elements to flow around it.
clear Prevents an element from being positioned next to a floated element on the same line.
